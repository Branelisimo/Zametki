ui.change_button_color()[исходный код]¶

Изменяет цвет кнопки между красным и синим.

Получает текущий цвет кнопки, меняет его на противоположный (красный → синий или синий → красный) и применяет новый цвет к кнопке.

Результат:

None

ui.show_message()[исходный код]¶

Показывает диалоговое окно с запросом подтверждения.

Выводит диалоговое окно с вопросом «Вы хотите продолжить?». В зависимости от ответа пользователя обновляет текст метки.

Результат:

None

ui.update_label()[исходный код]¶

Обновляет текст метки каждую секунду, пока счетчик меньше 10.

Глобальная переменная counter увеличивается на 1, и текст метки обновляется. Если счетчик достигает значения 10, функция прекращает вызывать себя.

Результат:

None
