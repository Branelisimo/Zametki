Модуль functions¶

functions.add_note()[исходный код]¶

Создает окно для добавления новой заметки и сохраняет её в список.

Открывает новое окно с текстовым полем и кнопкой «Сохранить». При нажатии на кнопку текст заметки добавляется в глобальный список notes вместе с текущей датой, после чего список обновляется.

Примечание

Если текст заметки пустой, отображается предупреждение.

functions.start_app()[исходный код]¶

Инициализирует и запускает основное окно приложения «Zametki».

Создает главное окно с кнопкой для добавления заметок и фреймом для их отображения. Запускает главный цикл приложения.

Global root:

Главное окно приложения.

Global notes_frame:

Фрейм для отображения заметок.

functions.update_notes_display()[исходный код]¶

Обновляет отображение списка заметок в основном окне.

Очищает текущий фрейм с заметками и отображает все заметки из списка notes в виде кратких меток. Каждая метка кликабельна для просмотра полной заметки.

См. также

view_note()

functions.view_note(index)[исходный код]¶

Отображает полное содержание заметки и предоставляет возможность её удаления.

Параметры:

index (int) – Индекс заметки в списке notes.

Результат:

None

Открывает окно с полным текстом заметки в режиме «только чтение» и кнопкой «Удалить». При нажатии на кнопку заметка удаляется из списка, а отображение обновляется.
